# TP SOA & SOAP — Bank Service

Service SOAP bancaire développé avec Spring Boot.

## Fonctionnalités

- GetAccount
- Deposit
- Withdraw
- Transfer

## Endpoint

http://localhost:8080/ws

WSDL :

http://localhost:8080/ws/bank.wsdl

## Postman

La collection Postman est disponible dans :

postman/TP_SOAP_Bank_Eya.postman_collection.json
